Selamun Aleyküm !

   Marmara Üniversitesi Yönetim Bilişim sistemleri 2.sınıf öğrencisiyim. Tasarımın temelini lise de Illustrator programı ile atmış olup, üniversitede de kendimi geliştirmeye devam ediyorum. Üniversite de Python öğrenip üzerinde bazı çalışmalar yaptıktan sonra, Web tasarım tarafında logo tasarlayarak, bunları hayatta pazarlamaya geçiriyorum. Portföyüme HTML ve CSS dillerini katarak kendimi daha da geliştirmeye çalışıyorum.
